# Comparison One — 4 Static Image Prompts

Campaign: Government-backed zero-interest SME loan awareness  
Use: ChatGPT Images / Hermes image generation / Canva background generation  
Format: 9:16 first, then crop to 4:5 and 1:1  
Important: generate text-free backgrounds, then add copy manually in Canva/Penpot/PIL.

## Reference From Canva Board

The Canva page is using ad-framework labels like:

- Industry Contrarian
- Listicle
- Founder
- Bandwagon Effect
- How you can X with/without Y
- The First 25

For Comparison One, the four strongest static angles are:

1. Eligibility / direct-response hook
2. Industry contrarian / big-bank bottleneck
3. Listicle / 3 checks before applying
4. Funding path map / with-or-without government program

Use a consistent dark premium fintech system: deep green, cream, muted gold, clean mobile-friendly overlays.

Add this to the start of every image prompt:

```text
Create a vertical 9:16 image for a premium Australian SME finance comparison ad. Deep forest green, cream, charcoal, and muted gold palette. ## Notes for Canva/Penpot Layout
```

- Add all text manually after image generation.
- Use C1 colours: primary `#074C3E`, dark `#03211B`, accent `#FCB650`, cream/white `#F5F6F7`.
- Keep the headline to 3–7 words.
- Use a small top pill label, one big headline, one support line, one gold CTA button.
- Avoid small comparison tables on the image. Use icons/cards instead.
- Do not add public compliance boilerplate to the design.
